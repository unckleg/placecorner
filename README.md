Components used 
--------------

The Place Corner v1.0 is powered with this components:

  * Symfony 3
  * Twig 
  * Doctrine ORM/DBAL
  * Vue.js 
  * Webpack - module bundler
  * Bootstrap 4, HTML 5, CSS3, JQuery

Whole code is written following latest and established PSR standards.
